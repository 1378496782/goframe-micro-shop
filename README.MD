# GoFrame 微服务电商项目架构

## 网关服务

### 🛒 Gateway-H5 (用户端)
- **用户服务 (User)**
    - 用户登录、注册
    - 修改密码
    - 获取用户信息

- **交互服务 (Interaction)**
    - 点赞/取消点赞
    - 评论/删除评论
    - 收藏/取消收藏
    - 查看互动内容

- **商品服务 (Goods)**
    - 商品列表展示
    - 商品详情查看

### 👨‍💼 Gateway-Admin (管理端)
- **管理员服务 (Admin)**
    - 管理员登录
    - 管理员注册

- **商品管理服务 (Goods)**
    - 商品增删改查
    - 商品详情管理
    - 商品数据图表分析

### 📁 Gateway-Resource (资源服务)
- **文件上传服务**
    - 图片上传与管理


## 项目启动
- 进入根目录app
- app包含全部服务和网关，进入目录启动main
- go run main.go
- 全部启动
