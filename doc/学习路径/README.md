## 记录原则

每轮学习只记录三类内容：

- 这轮解决了什么问题
- 核心设计为什么这么做
- API Fox / 数据库 / 日志怎么验证

不要把所有临时思考都塞到一个文件里。以后按模块拆开记，主线写在总路线里。
